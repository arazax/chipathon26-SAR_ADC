# StrongARM Comparator Wrapper Verification

## Block
`strongarm_comp.sch`

## Interface

Inputs:
- `vin_p`
- `vin_n`
- `rst_latch`
- `vdd`
- `vss`

Outputs:
- `out_p`
- `out_n`
- `comp_done`

## Operation

The StrongARM core operates in two phases:

- `rst_latch = 0`: reset/precharge phase. Both `out_p` and `out_n` are precharged high. `comp_done` returns to logic low.
- `rst_latch = 1`: evaluation phase. The comparator resolves according to the voltage difference between `vin_p` and `vin_n`.
